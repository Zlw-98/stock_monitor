## 系统简介

这是一套**估值驱动的自动定投辅助系统**，帮助您：
- 监控股票和ETF的估值状态
- 计算每月定投建议
- 检测组合偏离度并提醒再平衡
- 追踪分红日历
- 关注宏观数据对投资的影响

**核心特点：**
- 长期投资导向，不做短线交易
- 每周/每月定投决策辅助
- 估值+技术面+宏观面综合分析

## 快速启动

### 方式一：双击脚本启动

| 脚本 | 功能 |
|------|------|
| `start_dashboard.bat` | 仅启动 Dashboard 网页界面 |
| `start_monitor.bat` | 启动完整监控系统（含邮件推送） |

### 方式二：命令行启动

```bash
# 进入项目目录
cd d:\coding_projects\stock_monitor

# 启动 Dashboard
streamlit run app.py --server.headless=True

# 启动完整监控（含邮件推送）
python main.py
```

## Dashboard页面说明

### 1. 顶部总览
显示系统整体状态的关键指标。

| 指标 | 说明 |
|------|------|
| 总资产 | 根据持仓计算的账户总市值 |
| 本月定投金额 | 配置的每月定投预算 |
| 监控标的数 | 当前监控的股票/ETF数量 |

### 2. 持仓输入区域
手动输入或修改持仓数据：
- 持仓股数
- 当前价格
- 自动计算市值和比例

**操作：**
1. 输入各标的的持仓股数
2. 点击「保存持仓」将数据写入配置文件

### 3. 资产配置（饼图）
可视化展示当前持仓比例分布。

### 4. 估值状态面板
核心分析区域，显示各资产的估值情况：

| 字段 | 说明 |
|------|------|
| 标的名称 | 股票/ETF名称 |
| 估值指标 | PE分位 / 股息率 / PB |
| 数值 | 当前估值数据 |
| 估值判断 | 低估 / 正常 / 高估 |
| 状态颜色 | 绿色=低估，黄色=正常，红色=高估 |

**估值指标说明：**
- **标普500ETF**：PE分位（历史估值百分位）
- **红利ETF**：股息率（分红/价格）
- **长江电力**：股息率
- **交通银行**：PB（市净率）

### 5. 本月定投建议
根据估值和再平衡需求，生成当月定投建议：

| 字段 | 说明 |
|------|------|
| 标的 | 股票/ETF名称 |
| 建议金额 | 建议投入金额 |
| 优先级 | 执行顺序（星号越多越优先） |

**优先级规则：**
- ⭐⭐⭐⭐⭐ 最优先加仓
- ⭐⭐⭐ 正常配置
- ⭐⭐ 减少配置
- ⭐ 暂停/最低配置

### 6. 组合健康度
展示当前持仓与目标比例的偏离情况：

| 字段 | 说明 |
|------|------|
| 当前比例 | 实际持仓占比 |
| 目标比例 | 理想持仓占比 |
| 偏离 | 当前-目标的差值 |
| 状态 | 正常 / 需要调整 |

### 7. 标普500ETF溢价监控
监控境内ETF相对境外指数的溢价率：

| 溢价率 | 建议 |
|--------|------|
| > 3% | 暂停定投，转为现金管理 |
| 1% ~ 3% | 谨慎定投 |
| < 1% | 正常定投 |

### 8. 行情监控
显示今日各标的涨跌幅和预警状态。

### 9. 分红日历
显示即将到来的除权除息日：

| 字段 | 说明 |
|------|------|
| 股票名称 | 标的名称 |
| 除权除息日 | 分红到账日期 |
| 分红金额 | 每股/每份分红 |
| 持有份额 | 当前持仓数量 |
| 预计分红 | 分红金额 × 持有份额 |

## 每日邮件日报说明

每日15:10自动发送，包含以下部分：

### 一、市场概览
三大指数当日涨跌幅：
- 上证指数
- 恒生指数
- 标普500指数

### 二、估值分析
各标的核心估值数据：
- 当前价格
- 涨跌幅
- 股息率/PE分位/PB
- 数据来源和可信度

### 三、本月定投建议
最终执行定投方案，包含：
- 原始建议金额（基于估值）
- 调整后金额（考虑可信度）
- 执行优先级

### 四、组合健康度
当前持仓偏离目标的情况，标记需要调整的资产。

### 五、分红提醒
未来30天内的除权除息日预告，预计分红总额。

### 六、操作总结
执行策略建议：
- 哪些标的降低配置
- 哪些标的重点加仓
- 最终目标

### 七、宏观影响评估
未来30天重要宏观数据：

| 数据 | 影响 |
|------|------|
| CPI上升 | 利空股票，通胀压力 |
| CPI下降 | 利多股票，通胀缓解 |
| 非农强劲 | 可能压制降息预期 |
| FOMC鹰派 | 利空成长股 |
| FOMC鸽派 | 利多成长股 |

### 7. Backtest vs Live
回测收益与实盘收益对比。

## 定投策略逻辑

### 估值调整规则

**标普500ETF（PE分位）：**
- < 30% → 1.5倍定投（低估）
- 30% ~ 70% → 1倍定投（正常）
- > 70% → 0.5倍定投（高估）

**红利ETF（股息率）：**
- > 6% → 1.5倍定投（低估）
- 4% ~ 6% → 1倍定投（正常）
- < 4% → 0.5倍定投（高估）

**长江电力（股息率）：**
- > 4% → 1.2倍定投
- 3% ~ 4% → 1倍定投
- < 3% → 0.5倍定投

**交通银行（PB）：**
- < 0.7 → 1.5倍定投（低估）
- 0.7 ~ 1 → 1倍定投（正常）
- > 1 → 0.5倍定投（高估）

### 再平衡规则

**动态平衡触发开关：**
- 如果 |当前比例 - 目标比例| < 3% → 不触发再平衡
- 如果 |当前比例 - 目标比例| > 3% → 触发再平衡

**再平衡修正：**
- 超配 > 10% → 权重 × 0.5（减少）
- 低配 < -10% → 权重 × 1.5（增加）

### 可信度因子

根据数据可信度调整最终金额：

| 可信度 | 因子 |
|--------|------|
| > 80% | 1.0（正常） |
| 60% ~ 80% | 0.8 |
| 40% ~ 60% | 0.7 |
| < 40% | 0.5 |

## 常见问题

### Q: Dashboard打不开？
检查是否端口被占用，尝试：
```bash
# 停止现有进程
netstat -ano | findstr 8501
taskkill /PID <进程ID> /F
```

### Q: 没有收到每日邮件？
1. 检查垃圾邮件文件夹
2. 确认 `main.py` 是否在运行
3. 检查邮件配置是否正确

### Q: 估值数据都是N/A？
可能是网络问题或API限制，系统会自动使用默认值继续运行。

### Q: 如何修改每月定投金额？
编辑 `config/config.yaml`，修改 `monthly_investment` 的值。

### Q: 如何调整目标配置比例？
编辑 `config/config.yaml`，修改 `target_allocations` 部分。
